тестовое задание
/add - добавить ссылки и получить статусы
/report - сгенерировать PDF отчет
/shutdown - graceful shutdown сервера
пример запросов :
localhost:8080/add  
{
"links" : ["vk.com","youtube.com","music.yandex.ru"]
}
localhost:8080/report
{
"links_id" : [1]
}
localhost:8080/shutdown
без json
в данном задании реализованны все основные задачи
в качестве хранилища используются json файлы . при выключении сервиса реализуется graceful shutdown, с помощью которого в течении таймаута
программа не дает отправлять новые запросы , но она продолжает обрабатывать старые , запросы и выключается не теряя информацию , тк
при каждом запросе он сразу же отправлятся в processing , дабы не потерять его после выключение сервиса .graceful shutdown также ловит сигналы с ide и консоли .
вся логика прописанна в logic
в http вы найдете handlers , также я реализовал DTO И В cmd  находится мэйн с точкой входа . в данном ТЗ я показал умение использоваться
контекстами , горутинами , мьютаксами , http протоколами и владением знанием в написании архитектуры .
